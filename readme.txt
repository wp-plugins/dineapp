== Description ==

The DineApp Wordpress plugin allow restaurants to receive and manage reservations directly through their restaurant’s Web site or blog. 

DineApp is a free online reservation service provided by DineApp.com. By integrating with the most popular social networks (Facebook, Twitter, etc.), DineApp provides the easiest way for restaurants to manage reservations more effectively. The DineApp Wordpress plugin is a new service created by DineApp to help you get online reservations from your Wordpress site.


Features include:

* Booking widget, which allows customers to make reservation on your Wordpress site.
* Show restaurant availability on a given date
* Customized booking tab to show more information and recent restaurant events
* Adjust the layout of the booking widget to fit into the existing theme
* Integrate with DineApp iPhone app; get notified when there are new reservations!

It's free! 
Turn your Blog readers into Diners right now!


== Installation ==

To install DineApp Wordpress plugin:

1. Upload the dienapp folder to the /wp-content/plugins/ directory
1. Activate the DineApp plugin through the 'Plugins' menu in WordPress
1. Input your DineApp account to import your restaurant information (including restaurant name, quota setting, etc. It's totally free)
1. Configure the plugin by going to the DineApp menu that appears in your admin menu


== Frequently Asked Questions ==

= How do I check the reservations booked from this plugin? =
DineApp provide an iPhone app for you to manage your reservations. You can download this app at [DineApp at iTues Store](https://itunes.apple.com/us/app/dineapp-manage-restaurant/id568850083?ls=1&mt=8). This app enables restaurants to directly manage reservations from the most popular social networks (Facebook, Twitter, etc.), as well as their restaurant’s Web site or blog. 
Did I mention that this app is free, too?


= If I got too many reservations on one date, how do I stop new reservations from being made? =
You can stop new reservations from being made in the DineApp iPhone app. If you don't use this iPhone app, you can also set it up in [DineApp website](http://www.dineapp.com/?utm_campaign=dineapp_intro&utm_source=wp_plugin_directory&utm_medium=app_intro).


= Can I customized the title of the booking tab or sidebar widget? = 
Yes, you can customized the title in the dineapp admin panel.


= What languages are supported? = 
Currently this plugin only support English. We are currently working on Dutch, German, and more. If your language is not supported or if you want to help the localization, please contact us at service@dineapp.com.
